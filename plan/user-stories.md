# User Stories

1. As Jerry, I want to play a simple and rewarding slot machine game, so that I can relax, have fun, and enjoy the chance to win without dealing with complicated features.
2. As Serina, I want to play a visually exciting and customizable slot machine game, so that I can keep my streams entertaining and enjoy a fresh experience that matches my style.
3. As Alex, I want to quickly understand how the slot machine works through clear instructions and intuitive design, so that I can start playing immediately without confusion.
4. As Isabella, I want to track my wins, losses, and play history, so that I can manage my gameplay and make more informed decisions over time.
5. As Marcus, I want to set limits on my spending and playtime, so that I can enjoy the game responsibly without going over my budget.
6. As Haolong, I want a simple and responsive game interface, so that I can quickly understand what’s happening and focus on playing without confusion.
