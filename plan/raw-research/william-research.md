# Slot Machine App Research – [William Wang]
## Overview
This research explores how slot machine apps work, what makes them engaging, and what users expect. The goal is to identify key design principles that can guide the development of a simple, effective slot machine application.

## Domain Research
A slot machine is a chance-based game where players spin reels and receive rewards based on symbol combinations. The core gameplay is simple, but modern apps enhance it with animations, sound effects, and bonus features.

The main takeaway is that slot machines are not about strategy—they are about **experience, speed, and feedback**.

## Core Features
Common features in slot machine apps include:
- Spin button
- Reels and symbols
- Balance (credits) display
- Bet controls
- Win/loss feedback
- Sound effects and animations

These features form the basic gameplay loop: **bet → spin → result → feedback**.

## User Research
Users generally want:
- Simple controls
- Fast and responsive gameplay
- Clear display of balance and results
- Immediate feedback after each spin
  
Common issues include confusing interfaces, slow animations, and too many interruptions.
