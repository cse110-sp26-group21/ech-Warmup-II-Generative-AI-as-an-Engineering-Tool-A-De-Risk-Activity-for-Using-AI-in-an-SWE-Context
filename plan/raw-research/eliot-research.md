# Slot Machine Research Notes - Eliot

## Player Psychology and Engagement
Keeping players engaged with the game is a very important goal when creating a slot machine. While themes and animations play a roll in retaining user focus, there are certain features that can incentivize users to continue playing. 
Some are such as the following:
- Near-miss effect: By getting the user close to winning, users are urged to play more and chase the jackpot
- Variable reward timings: Unpredictably and occasianally awarding wins to the user
- Losses are disguised: Celebrating losing payouts as small wins trick player to believe that they aren't losing money
- Progressive jackpot: A jackpot that increases as a user plays more

## Game Mechanics
- Payout lines: Can have wins not only horiontally, but also diagonally
- Symbol Weighting: Rarer symbols can lead to larger payout
- Multipliers: Have chance to earn larger payouts with same bet
- Wild Symbol: Can be a substitute for other symbols

## Platform Consideration
With everyone having a phone or some other kind of electronic device these days, the slot machine game needs to be playable on multiple interfaces
- Should display correctly on both mobile and desktop screens
- Touch interactions if mobile
- Screen size constraints and maintain correct proportions
- Smooth performance on all platforms

## Display
Information and features that the user needs to play the slots game
