# Slot Machine Notes

## What is a Slot Machine?
A slot machine is a game where a player places a bet, spins the reels, and receives a result based on the symbol combination shown after the spin. The goal is usually to match symbols in a winning pattern and earn a payout.

## Common Terms
- **Reels**: The spinning columns in the machine
- **Symbols**: The icons that appear on the reels
- **Spin**: The action that starts the game round
- **Bet**: The amount the player risks for one round
- **Credits**: The player’s current balance or points
- **Payout**: The reward received from a winning combination
- **Paytable**: A chart that explains what symbol combinations win

## Common Features in Slot Machine Games
- Spin button
- Bet increase / decrease buttons
- Credit display
- Win/loss result message
- Restart or reset button
- Sound effects
- Reel spin animation
- Paytable or help section
- Current bet display
- Last result display

## User Needs
Players may want:
- Simple and clear controls
- Fast understanding of game rules
- Immediate feedback after each spin
- Easy-to-read credits and bet amounts
- Fun visual effects without too much clutter
