# AI Plan

## Goal
Our goal is to use a generative AI coding assistant to help us build a better slot machine game while evaluating how useful AI is in a software engineering workflow.

## Chosen Tool
Claude Code

## Why We Chose This Tool
We chose Claude Code because it is useful for generating code, explaining structure, and helping with iteration.

As a computer science student and gamer, Haolong Chen is interested in using AI tools to speed up development while still maintaining code quality.

## Planned Strategy
- Break tasks into small steps
- Use clear and specific prompts
- Review AI-generated code carefully
- Use linting and testing frequently
- Record all AI usage in the log

## Quality Controls
- Linted code
- Documented code
- Unit tests
- Modular structure

## Human Editing Policy
We will only manually edit code if AI fails to fix an issue, and we will record it in the log.
